Etiquetas en HTML

Dentro de HTML encontramos un monton de eqtiquetas, trataré de resumir las que más se suelen ocupar con el día a día explicando en breve su uso y como se declaran en el HTML.

Las etiquetas las podemos dividir en diversos grupos entre los que podemos encontrar los siguentes:

  1. Etiquetas de Meta Datos
  2. Etiquetas de Sección
  3. Etiquetas de Contenido
  4. Etiquetas de Texto en Linea
  5. Contenido Embebido
  6. Tablas
  7. Formularios

Etiquetas de Meta Datos:

  1. <head> </head>

    Es la etiqueta que nos permite colocar dentro toda la colección de meta datos.

    
<head>

</head>
  2. <link>

    Especifica la relación entre el documento actual y un recurso externo.

    
<link rel="stylesheet" href="css/style.css">

    Esta etiqueta es de solo apertura por lo tanto no hay que indicar el cierre y hace uso de dos atributos el primero es rel="" al que se le coloca un valor que indica la relación que va a tener el archivo externo con el documento HTML y el segundo href="" a este se le coloca la ruta de donde va a extraer este archivo.
    En este caso la relación que tiene es una hoja de estilos (stylesheet).

  3. <meta>

    Sirve para aportar información sobre el documento.

    
<meta charset="UTF-8">

    Esta etiqueta también es de solo apertura y generalmente tiene dos atributos name="" y content="" pero existen excepciónes como la que muestra el ejemplo que lo que indicamos con ese meta es el tipo de formato de codificación de caracteres tiene nuestro sitio web que en este caso es UTF-8 que nos permite que lea los caracteres especiales del lenguaje español como con las letras con tildes.
    Pero con esta etiqueta también se le puede colocar el autor de la página, la descripción de la página, entre otras.

  4. <title> </title>

    Indica el título del documento.

    
<title>Mi primera página web</title>
  5. <style> </style>

    Es el elemento encargado de indicar la información de estilo.

    
<style>
    .content{
        width: 90%;
        margin: auto;
    }
</style>

    Dentro de esta etiqueta se coloca codigo CSS para darle estilos a nuestro sitio web, durante mucho tiempo se consideraba una mala practica colocar esto dentro del mismo HTML ya que se llenaba de 2 lenguejes diferentes el HTML. Pero ahorita en la actualidad se sule poner en el mismo HTML cuando hablamos de WEB COMPONENTS que es una forma de crear y alamacenar pequeños elementos del sitio web para despues poderlos reutilizar de manera mas sencilla.
    Nota.- (Pero si no vas a crear ningun WEB COMPONENT no te recomiendo que lo coloques dentro del HTML).

Contenido Embebido

  1. <img>

    Se utiliza para colocar imagenes. Esta etiqueta es de solo apertura y se lo colocan 2 atributos el primero es src="" que en el se coloca la ruta o dirección de donde sera importada la imagen y el segundo es alt="" aqui se coloca una pequeña descripción de la imagen (si la imagen carga sin ningun problema lo que se coloque dentro del atributo no se mostrará pero en caso contrario lo que se visualizará sera el valor del atributo).

    
<img src="http://lorempixel.com/300/200" alt="Esta imagen es aleatoria, es traida de Lorempixel.">
  2. <audio></audio>

    Se utiliza para importar un audio al sitio web. Esta etiqueta al igual que la de <img> tiene el atributo src="" para colocar la ruta o dirección de donde obtendremos el audio. A la etiqueta se le pueden agregar atributos adicionales para controlar el audio que se va a insertar como son:

    • loop.- Nos sirve para repetir el audio automaticamente infinitas veces.
    • autoplay.- Sirve para reproducir de manera automatica el audio al momento de cargar el sitio web.
    • controls.- Sirve para agregar los controles basico para controlar el audio.
    
<audio src="mi_audio.mp3" controls></audio>
  3. <video></video>

    Se utiliza para importar un video al sitio web. Al igual que el resto de etiquetas de esta sección debemos de agregar el atributo src="" para indicarle de donde queremos que nos importe el video. Y al igual que la etiqueta de <audio></audio> podemos agregar atributos adicionales para un mejor control del video.

    • loop.- Nos sirve para repetir el video automaticamente infinitas veces.
    • autoplay.- Sirve para reproducir de manera automatica el video al momento de cargar el sitio web.
    • controls.- Sirve para agregar los controles basico para controlar el video.
    
<video src="mi_video.mp4" controls></video>

Formularios

Los formularios son la herramienta que nos permite solicitarle información al usuario y despues poderla procesar con algun lenguaje de programación del lado del servidor.

  1. <form></form>

    Es la etiqueta principal para definir un formulario. Esta etiqueta tiene 2 atributos principales action="" que indica a que documento enviará los datos del formulario para despues ser procesados desde ese documento y method="" que inica el metodo de envio de los datos.

    
<form action="php/datos.php" method="post">
    
</form>
  2. <label></label>

    Es un elmento para mostrar información, dicho de otro modo es una de las etiquetas que muestra información. Se le puede colocar un atributo para asignarlo a un elemento especifico que se hace mediante el atributo for="" donde se coloca alguna palabra clave y el elemento con el que se le valla a asosiar debe de contener la misma palabara clave en el atributo id.

    
<form action="php/datos.php" method="post">
    <label for="texto">Soy una etiqueta</label>
    <input type="text" id="texto">
</form>
  3. <input type"text">

    Crea una caja de texto que permite ingresar texto.

    
<form action="php/datos.php" method="post">
    <input type="text" name="texto">
</form>
  4. <input type"email">

    Crea una caja de texto que permite ingresar un correo electronico con la facilidad de que internamente el navegador valida que el usuario ingrese un correo valido.

    
<form action="php/datos.php" method="post">
    <input type="email" name="email">
</form>
  5. <input type"password">

    Crea una caja de texto que permite ingresar un correo electronico con la facilidad de que internamente el navegador valida que el usuario ingrese un correo valido.

    
<form action="php/datos.php" method="post">
    <input type="password" name="password">
</form>
  6. <input type"checkbox">

    Crea un elemento para poder seleccionar o deseleccionar el elemento.

    
<form action="php/datos.php" method="post">
    <input type="checkbox" name="checkbox" value="checkbox">
</form>
  7. <input type"radio">

    Crea un elemento para poder seleccionar pero la diferencia con el checkbox es que puedes crear un grupo de estos elementos para que solo puedas selecionar uno de muchos.

    
<form action="php/datos.php" method="post">
    <!-- Radio solo -->
    <input type="radio" name="radio" value="radio">

    <!-- Grupo de Radios -->
    <input type="radio" name="genero" value="hombre">
    <input type="radio" name="genero" value="mujer">
</form>

    Para indicar que tenemos un grupo de radios basta con ponerles el mismo nombre a los radios que formarán parte del grupo.

  8. <input type"tel">

    Crea una caja de texto, a nivel de una comptadora no se ve ninguna diferencia pero en un smarphone al dar clic abre el teclado de solo numeros.

    
<form action="php/datos.php" method="post">
    <input type="tel" name="tel">
</form>
  9. <input type"date">

    Crea una caja para poder ingresar una fecha. (Dependiendo del navegador pude tener efectos propios del mismo, por ejemplo Chrome muestra un calendario de donde puedes seleccionar la fecha).

    
<form action="php/datos.php" method="post">
    <input type="date" name="date">
</form>
  10. <input type"reset">

    Crea un boton que permite limpiar todos los inputs.

    
<form action="php/datos.php" method="post">
    <input type="text" name="texto">
    <input type="password" name="password">
    <input type="radio" name="genero" value="hombre">
    <input type="radio" name="genero" value="mujer">

    <input type="reset" name="reset" value="Resetear Valores">
</form>
  11. <input type"time">

    Crea un caja para poder incrustar una hora.

    
<form action="php/datos.php" method="post">
    <input type="time" name="time">
</form>
  12. <input type"number">

    Crea una caja con un spin para poder poner un numero e ir aumentado o disminuyendo e valor con el spiner.

    
<form action="php/datos.php" method="post">
    <input type="number" name="number">
</form>
  13. <input type"range">

    Crea un elemento tipo barra de volumen que permite seleccionar un valor.

    
<form action="php/datos.php" method="post">
    <input type="range" name="range">
</form>

    Al <input type="range" name="range"> se le puede indicar el numero de donde va a partir con el atributo min="" y tambien indicarle un valor maximo con el atributo max="", lo puedes personalizar aun más para decirle de cuanto en cuanto va a ir avanzando con el atributostep="" y por ultimo le puedes indicar el valor inicial con el atributo value=""

    
<form action="php/datos.php" method="post">
    <input type="range" min="0" max="100" step="20" value="20" name="range">
</form>
  14. <input type"color">

    Crea una caja que muestra un color y al dar clic te permite seleccionar algun otro color.

    
<form action="php/datos.php" method="post">
    <input type="color" name="color">
</form>
  15. <input type"file">

    Crea un boton que permite seleccionar algun archivo para poderlo enviar.

    
<form action="php/datos.php" method="post">
    <input type="file" name="file">
</form>
  16. <input type"submit">

    Crea un boton que su función será enviar la información del formulrio.

    
<form action="php/datos.php" method="post">
    <input type="submit">
</form>

    Se le puede colocar el atributo value="" para personalizar el texto que se coloca en el botón.

    
<form action="php/datos.php" method="post">
    <input type="submit" value="Ingresar">
</form>
  17. <input type"image">

    Crea un boton a partir de una imagen que se tiene que especificar la ruta de donde sera obtenida y la función es enviar los datos.

    
<form action="php/datos.php" method="post">
    <input type="image" src="http://img.pngget.com/clip1/403kcajfkhl.png">
</form>
  18. <button></button>

    Crea un boton que su función será enviar la información del formulrio.

    
<form action="php/datos.php" method="post">
    <button>Mi boton</button>
</form>
  19. <select></select>

    Crea una especie de combo que permite seleccionar una opción de una lista desplegable.

    
<form action="php/datos.php" method="post">
    <select name="select">
        
    </select>
</form>
  20. <option></option>

    Con esta etiqueta podemos agregar items al combo.

    
<form action="php/datos.php" method="post">
    <select name="select">
        <option value="Carro">Carro</option>
        <option value="Moto">Moto</option>
        <option value="Avión">Avión</option>
        <option value="Ferri">Ferri</option>
    </select>
</form>
  21. <optgroup></optgroup>

    Nos sirve para agrupar un conjunto de opciones (items) del combo.

    
<form action="php/datos.php" method="post">
    <select name="select">
        <optgroup label="Terrestres">
            <option value="Carro">Carro</option>
            <option value="Moto">Moto</option>
        </optgroup>
        <optgroup label="Aereos">
            <option value="Avión">Avión</option>
        </optgroup>
        <optgroup label="Acuaticos">
            <option value="Ferri">Ferri</option>
        </optgroup>
    </select>
</form>

    A la etiqueta se le coloca el atrubuto label="" para darle un nombre a la categoría que contendrá nuestras opciones.

  22. <textarea></textarea>

    Es como un input de tipo texto la diferencia es que el textarea te permite escribir en varias lineas.

    
<form action="php/datos.php" method="post">
    <textarea name="textarea"></textarea>
</form>

    Se puede limitar el numero de caracteres que el usuario puede escribir en el <textarea></textarea> agregando la propiedad maxlength=""

    
<form action="php/datos.php" method="post">
    <textarea name="textarea" maxlength="10"></textarea>
</form>
  23. <fieldset></fieldset>

    Sirve para agrupar elementos dentro del formulario.

    
<form action="php/datos.php" method="post">
    <fieldset>
        <label for="texto">Soy una etiqueta</label>
        <input type="text" id="texto name="text">
    </fieldset>
</form>
  24. <legend></legend>

    Se utiliza para colocarle un titulo al grupo de campos.

    
<form action="php/datos.php" method="post">
    <fieldset>
        <legend>Datos del Inicio de Sesión</legend>
        <input type="text" name="texto">
        <input type="password" name="password">
    </fieldset>
    <fieldset>
        <legend>Genero</legend>
        <input type="radio" name="genero" value="hombre">
        <input type="radio" name="genero" value="mujer">
    </fieldset>
</form>

Todo elemento del formulario que valla a recibir información del usuario debe de tener un atributo importante que es el name="" sin este atributo el formulario no envia ese valor y al ocurrir eso no podremos rescatar la información del lado del servidor para ser procesada.
Hay elementos que también requieren el atributo value="" esto porque los usuarios no puede colocar de forma manual el valor pero internamente nosostros podemos indicarle un valor especifico que al ser procesado lo identifiquemos con nuestro valor. Y los elementos que llevan ese atributo son todos aquellos donde el ususario no puede indicar la información directamente él, osea que el no la escribe.

Exiete un atributo más que podemos agregarle a los elementos de un formulario, en especifico los campos de tipo texto que es placeholder="" que proporciona un texto como ayuda a la información que debes escribir dentro de ellos y el usuario al escribir texto desaparece esa ayuda.


<form action="php/datos.php" method="post">
    <fieldset>
        <legend>Datos del Inicio de Sesión</legend>
        <input type="text" name="texto" placeholder="Nombre:">
        <input type="password" name="password" placeholder="Contraseña:">
    </fieldset>
    <fieldset>
        <legend>Genero</legend>
        <input type="radio" name="genero" value="hombre">
        <input type="radio" name="genero" value="mujer">
    </fieldset>
</form>
