ESTRUCUTRA DE UN DOCUMENTO HTML

Todo docuemento HTML debe iniciar por el DOCTYPE y se escribe:


<!DOCTYPE html>

*Nota.- El <!DOCTYPE html> le indica al navegador con que tipo de lenguaje va a interpretar que en este caso es HTML.


Después de indicarle el tipo de lenguaje que va a reconocer el navegador tenemos que dar inicio a nuestro HTML y eso lo hacemos con la etiqueta de <html> y cabe mecionar que esta es una etiqueta de apertura y cierre por lo tanto hay que colocarla asi:


<!DOCTYPE html>
<html>

</html>

*Nota.- La etiqueta <html> </html> representa la raiz de un documento HTML


Dentro de la etiqueta <html> </html> tenemos que colocar 2 etiquetas importantes que son el <head> </head> y el <body> </body> que tambien son etiquetas de apertura y cierre.
La etiqueta <head> </head> representa una coleccion de metadatos para el documento. Dicho esto de otra manera es la parte del codigo que le indica información al navegador y motores de busqueda sobre nuestra página o sitio web.
La etiqueta <body> </body> representa el contenido del documento. Dicho de otra manera es la parte del codigo que contendrá todo el contenido de nuestra página o sitio web.
Ambos se escriben de la siguente manera en nuestro documento:


<!DOCTYPE html>
<html>
	<head>

	</head>
	<body>
	
	</body>
</html>

Dentro de la etiqueta head se pueden colocar metadatos como el tipo de codificación que usaras en el contenido.
Una etiqueta importante dentro de <head> es <title>
Esta etiqueta representa el titulo o nombre del documento. Los autores deberian usar titulos que identifiquen sus documentos incluso cuando ellos sean usados fuera de contexto, por ejemplo dentro de una histria del autor o marcadores, o dentro de resultados de busqueda. El titulo del documento es a menudo diferente de su primer encabezado, ya que el primer encabezado no esta solo cuando esta fuera de contexto. Y se coloca de la siguente manera:


<!DOCTYPE html>
<html>
	<head>
		<title>Mi primera página web</title>
	</head>
	<body>
	
	</body>
</html>
